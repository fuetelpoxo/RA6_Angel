## Uso


El sitema de veterinario incluye lo siguiente 

+ solicitud gestio
+ acceso y consulta
+ comunicacion entre opacientes
+ administracion del personal 
+ revision de informes
+ gestion de facturas


Para utiizar la aplicacion

|**Usuario**    | **funcionalidad**|
